One-time setup. Paste your Firebase project credentials below. These are saved in your browser only.
How to set up Firebase
1
Go to console.firebase.google.com and sign in.
2
Click Add project, name it ipl-fantasy-2026, disable Analytics, create it.
3
Go to Build → Realtime Database → Create database → choose region → test mode.
4
Project Settings → Your apps → </> Web → register → copy firebaseConfig values into the fields.
5
Click Save & Connect, then go to Owners to create your first admin account.
